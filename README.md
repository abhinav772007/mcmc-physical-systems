# Statistical Analysis of Physical Systems using Monte Carlo and MCMC

This repository contains assignments, simulations, and numerical experiments completed as part of the **PAAC Monte Carlo and MCMC Project** conducted at the Indian Institute of Technology Roorkee.

The project explores stochastic simulation techniques, probabilistic modeling, Markov processes, and Monte Carlo based methods for analyzing physical systems.

---

## Topics Covered

### Week 1
- Basics of Python
- Introduction to Thermodynamics

### Week 2
- Fundamentals of Probability and Statistics
- Random Variables and Distributions

### Week 3
- Monte Carlo Methods
- Random Sampling Techniques
- Numerical Estimation

### Weeks 4–5
- Markov Chains
- Transition Probabilities
- Sampling Techniques

### Week 6
- Markov Chain Monte Carlo (MCMC)
- Metropolis-Hastings Algorithm
- Gibbs Sampling

### Weeks 7–8
- Simulation of the 2D Ising Model
- Spin Configurations
- Energy and Magnetization Analysis
- Phase Transition Behavior

### Weeks 9–10
- Final Assignment and Project Work
